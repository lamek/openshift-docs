= Installing Metering
include::modules/common-attributes.adoc[] 
:context: metering 

toc::[]

On OpenShift 4.x clusters the Metering Operator is available through Operator
Hub and can be installed using the Operator Lifecycle Manager (OLM). If you're
using an earlier version of OpenShift or a Kubernetes cluster without OpenShift,
please use our upstream docs to install, configure, and use the Metering
Operator.
